👤 Accounts & Cross-Device Sync (one-time setup)

Set this up once so every user gets a clean sign-in screen, and everyone's memories/settings sync across their devices. Powered by Supabase (free, no credit card).

One-time setup (2 minutes): go to supabase.com → New Project (free) → in Project Settings → API, copy your Project URL and anon public key, paste them below.

Then open the SQL Editor in Supabase, paste this, and click Run — this creates your private data table:

create table faith_user_data ( user_id uuid primary key references auth.users(id), memories jsonb, profile jsonb, groq_key text, chats jsonb, updated_at timestamptz ); alter table faith_user_data enable row level security; create policy "own data" on faith_user_data for all using (auth.uid() = user_id);
Signed in as . Your data syncs automatically across every device you sign into.
